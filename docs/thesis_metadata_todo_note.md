# Thesis Metadata TODO Note

Bu dosya, SAÜ FBE frontmatter üretiminde kullanılan docs/thesis_metadata.yaml dosyasındaki geçici TODO alanlarını açıklar.

## Bilinçli Olarak Geçici Bırakılan Alanlar

Aşağıdaki alanlar şu aşamada bilinçli olarak TODO bırakılmıştır:

- author
- student_id
- advisor
- jury

## Gerekçe

Bu alanlar nihai teslim öncesinde gerçek öğrenci, danışman ve jüri bilgileriyle doldurulacaktır. Mevcut dosya, içerik geliştirme ve biçimsel taslak kontrolü amacıyla kullanılmaktadır.

## Build Politikası

Metadata içinde TODO alanları olduğu sürece SAÜ FBE ana tez build işlemi aşağıdaki parametreyle çalıştırılabilir:

--allow-todo-metadata

Nihai teslim adayı üretileceği zaman bu parametre kullanılmamalı ve metadata kalite kontrol raporunda TODO count = 0 olmalıdır.
